# 📚 BookStore Management System

A full-stack MERN BookStore Management System developed using the MERN Stack as part of the SmartBridge Internship Program.

---

## 🚀 Features

- User Authentication (Login & Register)
- Book Management
- Shopping Cart
- QR Code Payment
- Admin Dashboard
- Responsive User Interface
- MongoDB Atlas Database Integration

---

## 🛠️ Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Bootstrap

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/darshini241/BookStore.git
```

### Install dependencies

```bash
npm install
```

### Start Backend

```bash
npm start
```

### Start Frontend

```bash
npm run dev
```
